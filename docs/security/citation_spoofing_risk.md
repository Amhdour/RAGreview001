# Citation Spoofing Risk

| Spoofing path | File/path evidence | What is confirmed | What is unverified | Potential impact | Later validation method |
| ------------- | ------------------ | ----------------- | ------------------ | ---------------- | ----------------------- |
| Document text pretending to be citation | `backend/onyx/chat/citation_processor.py`; `backend/onyx/file_processing/html_utils.py` | Citations are parsed from plain text patterns while HTML content is cleaned upstream | Whether crafted document text can still mimic trusted citation syntax in user-visible output | Misleading citation appearance | Run controlled content-fuzzing against citation rendering |
| Source title/URL spoofing | `backend/onyx/connectors/models.py`; `backend/onyx/connectors/web/connector.py`; `web/src/components/search/results/Citation.tsx` | Title and URL fields are passed through display paths | Whether malicious or stale metadata can spoof a trusted source label | User trust in the wrong source | Validate display labels against canonical source IDs in runtime tests |
| Metadata spoofing | `backend/onyx/connectors/models.py`; `backend/onyx/db/models.py`; `web/src/refresh-components/buttons/source-tag/SourceTagDetailsCard.tsx` | Metadata is persisted and rendered in source cards | Whether the metadata is sanitized against adversarial values | Misleading source cards or tooltips | Test rendering with adversarial metadata payloads |
| Markdown/HTML spoofing | `backend/onyx/chat/citation_processor.py`; `backend/onyx/file_processing/html_utils.py` | Markdown and HTML are both present in the source path | Whether crafted markup can visually impersonate citations | Source confusion or clickjacking-like deception | Fuzz markdown and HTML handling in renderers |
| Citation marker spoofing | `backend/onyx/chat/citation_processor.py`; `backend/tests/unit/onyx/chat/test_citation_processor.py` | Marker parsing recognizes multiple bracket variants | Whether malformed markers or adjacent text can still spoof a citation | Users may infer false evidence linkage | Execute the marker-parsing tests and malformed-marker runtime cases |
| Frontend rendering spoof risk | `web/src/components/search/results/Citation.tsx`; `web/src/refresh-components/buttons/source-tag/SourceTag.tsx` | The UI renders labels, tooltips, and details cards from source metadata | Whether the UI defends against deceptive labels from upstream data | The display layer may amplify source confusion | Verify frontend rendering with adversarial source metadata |
