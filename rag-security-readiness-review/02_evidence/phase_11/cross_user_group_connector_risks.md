# Cross-User / Group / Connector Risks

| Risk path | Related users/groups/connectors | File/path evidence | What is confirmed | What is unverified | Potential impact | Later validation method |
| --------- | ------------------------------- | ------------------ | ----------------- | ------------------ | ---------------- | ----------------------- |
| Cross-user retrieval | User email, public ACLs | `backend/onyx/access/access.py`, `backend/onyx/db/document_access.py` | User ACLs are built from the current user email plus public access. | Whether another user's private content can never surface at runtime is unverified. | Unauthorized document exposure between users. | Negative search test for a known forbidden document ID. |
| Cross-group retrieval | User groups, external user groups | `backend/ee/onyx/access/access.py`, `backend/onyx/access/models.py` | EE ACLs can include group names and external group IDs. | Group-sync completeness and timing are unverified. | Exposure across group boundaries. | Validate search results against a user missing the granting group. |
| Cross-tenant/workspace retrieval | Tenant ID, workspace-adjacent context | `backend/onyx/context/search/pipeline.py`, `backend/onyx/document_index/opensearch/search.py`, `backend/onyx/document_index/vespa/shared_utils/vespa_request_builders.py` | Tenant ID is injected into retrieval filters when multi-tenant mode is enabled. | Explicit workspace-scoped retrieval was not identified in reviewed source. | Exposure across tenant boundaries if filters fail. | Verify tenant-separated search results in a live multi-tenant environment. |
| Cross-connector retrieval | Connector credential pairs, source types | `backend/onyx/db/document_access.py`, `backend/ee/onyx/external_permissions/post_query_censoring.py` | Document access and censoring are connector/source aware. | Whether all connector/source combinations enforce identical behavior is unverified. | Exposure between sources with different ACL models. | Validate per-source result sets for a user lacking one connector. |
| Connector permission drift | Sync timestamps, stale external groups | `backend/onyx/db/connector.py`, `backend/ee/onyx/db/external_perm.py`, `backend/ee/onyx/background/celery/tasks/external_group_syncing/tasks.py` | The source tracks sync timestamps and stale rows. | Drift window and recovery timing are unverified. | Temporary exposure after source permission changes. | Compare live source permissions to live search results after a permission change. |
| Inaccessible source / citation exposure | Search results, citations | `backend/onyx/chat/citation_utils.py`, `web/src/components/search/results/Citation.tsx` | Returned search documents feed citation rendering. | Whether inaccessible source metadata can still surface in the UI is unverified. | Leakage of source names, titles, or links. | Validate citation rendering for a forbidden source. |
